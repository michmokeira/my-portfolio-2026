# Portfolio Website

This repository contains the source code for my personal portfolio website.

The site is designed to showcase selected projects across **software development and UI/UX design**, highlighting both the technical implementation and the design process behind each project.

## Overview

The portfolio focuses on:

* Full-stack and frontend development projects
* UI/UX design case studies
* Interactive and motion-driven web interfaces

The goal of the site is to present projects in a way that communicates **problem solving, design thinking, and technical execution**.

## Tech Stack

* React
* Vite
* Tailwind CSS
* Framer Motion
* GSAP
* Lenis
* React Router

## Status

This portfolio is currently being redesigned and rebuilt.

The repository reflects the **active development version** of the site.
